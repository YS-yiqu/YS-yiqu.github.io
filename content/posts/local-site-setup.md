## 个人理解「答：」

1. 类似于文档模板一样的东西，改改就能用，可以叫作「框架」
2. 使用 git 工具
3. bash 是命令行环境的一个叫法，打开后就是命令行的操作
4. 一种编程语言的运行环境「JavaScript」
5. 有一个先决条件，Node.js 需要安装，因为 npm 这类似于一个函数的东西，在 Node.js 内置；其次是网络问题，查询 GPT 后给出一个镜像地址

   ```bash
   npm config set registry https://registry.npmmirror.com/
   npm install -g hexo-cli
   ```

6. 这里的复制按钮，是「Shift」+「Ins」的组合键，可能是为了防止误触？
7. 貌似是在这个地址下进行文件的下载相关操作
8. 就是一个写代码加上编译的一个程序，「开发工具」。因为参考的这篇文章是使用这个的，就按照这个思路来做
